1. 網頁架構圖（簡述）

架構說明：

  • Header（導覽） — 全站導航與基本資訊
  • Main（主內容） — About / Media（影片、照片） / Stats（統計圖表）
  • Aside（側欄） — 同學連結、快速操作
  • Footer（頁尾） — 著作權或備註

以下是一個簡單的架構圖（Mermaid 格式供老師參考）：

graph TD
  A[Header]
  B[Main]
  C[Aside]
  D[Footer]
  A --> B
  A --> C
  B --> D
  C --> D
  B -->|contains| E[Media]
  B -->|contains| F[Stats]

6. 上架說明（快速步驟）

  1. 使用 GitHub：建立一個 repo，把此檔案（index.html）放進 repo 的 root，然後在 repo 的 Settings → Pages 啟用 GitHub Pages（branch = main，/ root）。
  2. 或使用 Netlify / Vercel，直接 drag & drop 或連結 GitHub repo，設定部署即可。
  3. 也可將檔案上傳到任何支援靜態頁面的主機（FTP / S3 / 靜態託管）。

提示：若要讓圖片使用雲端圖床，建議使用 GitHub repo 的 /assets 目錄或可信任的圖床（例如你自己的 S3、Imgur 等）。
